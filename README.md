# web.grindr.com unlock

> [!NOTE] Does not work in Chrome.

Simple Firefox extension/add-on to unlock the web client on web.grindr.com for free users. Does not work in Chrome. Presented to you by [Open Grind](https://opengrind.org) project.

## Usage

1. Download the latest release from the [releases page](https://git.opengrind.org/open-grind/grindr-web-unlock/releases)
2. In Firefox, go to `about:addons` and click gear icon
3. Select "Install Add-on From File..." and select the downloaded `.xpi` file
